# 1.doc.merchants

Documentos em HTML (e outros) para enviar aos clientes/merchants, explicando integrações com Yuno.

**Convenção:** todo documento em HTML que explicar integração (PayPal, outros métodos, SDK Lite, Full, Direct API, etc.) deve ser colocado nesta pasta.

**Padrão visual:** todos os docs desta pasta seguem o mesmo padrão (fundo branco, código inline colorido sem caixa, blocos em quadrante preto com syntax highlight Cursor Dark, PDF-friendly). Referência: `paypal-integracao-yuno.html`. Regra do projeto: `.cursor/rules/doc-merchants-html.mdc`.

## Documentos

- **paypal-integracao-yuno.html** — Integração PayPal com SDK Lite, SDK Full e Direct API: possibilidades e como integrar em cada cenário (PT).
- **paypal-integracao-yuno.en.html** — Same content in English (PayPal integration with SDK Lite, SDK Full & Direct API — technical guide for developers).
